1с-Битрикс:Свободный старт.Неофициальная редакция. Версия: 0.1 бета.

[Описание]
Не требует покупки лицензии. 
Полностью бесплатная редакция.
Оптимизированное ядро.
Быстро работает.

[Установка]
1. Импортировать файл freetrix_start.sql в вашу базу. (utf8 кодировка базы и всего сайта!)
2. Загрузить файлы на хостинг (например в /www )
3. В freetrix/php_interface/dbconn.php и freetrix/.settings - прописать ваш host,login,password от базы

Доступы админа на сайт:
Логин: admin , пароль: qwerty2013

Далее меняйте себе пароль для админа, настраивайте шаблон и т.п

[Ограничения редакции]
- Установка модулей только вручную (не через маркетплэйс): 
- Не поддерживает обновлений
- Не поддержвает кеширование (убрано на уровне ядра)
- Не поддерживает облако битрикс
